# Clean Home Services

A modern, engaging static webpage for a residential cleaning business in the UK.

## Features
- Service information
- Pricing display (£13/hour for most services, £17/hour for post-construction, 2-hour minimum)
- Booking form with date and time selection
- Testimonials
- About us
- Contact information

## How to Run Locally
Open `index.html` in a web browser.

## Deployment
This static site can be deployed to any web hosting service. Recommended options:

### GitHub Pages
1. Push the repository to GitHub.
2. Go to repository Settings > Pages.
3. Select "Deploy from a branch" and choose `main` branch.
4. The site will be available at `https://<username>.github.io/<repo-name>/`.

### Netlify
1. Sign up for a Netlify account.
2. Drag and drop the project folder or connect your Git repository.
3. Netlify will deploy the site automatically.
